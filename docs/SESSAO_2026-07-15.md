# Sessão 2026-07-15

## Teste de segurança - allowFrom validado

Simulado webhook com número fora da lista (15551234567)
+ assinatura HMAC válida → resultado: bloqueado
corretamente (log "not authorized"), zero processamento
pelo agente.

Controle negativo: mesmo payload com assinatura inválida
→ rejeitado com 403 antes mesmo de chegar no allowFrom,
confirmando que as duas camadas de segurança (assinatura
+ allowlist) funcionam de forma independente.

Conclusão: proteção contra mensagens não autorizadas
está funcionando como projetado.
